EC_BASE

This dictionary is used for script-specific configuration that does not include algorithm changes. It includes options like what function to use as the main test function, whether to send FIS events, etc. This specific dictionary holds default values, so that userscript will always work. The actual values for each script should be stored in that script's TestName.py.

CommonPAEData()

source code  
Returns a set of common required data for PAE_PLUS tables in string format.
Common parameters - SERIAL_NUM, TUID, RUN_ID, CHAMBER, SLOT, EVENT_DATE,
                    AUTH, SUB_TEST and SUPRA_SUBTEST (9 fields total).
                    Newline characters are inserted in between fields.

eventDate: string; Last event date set. Can optionally supply a specific time (i.e. test start or time of failure)

SendTestInit(ESG_test=0)

source code  

Sends test init message during Start Drive
Re-written with string substitution instead of addition for easier readability

ESG_test - boolean = 0; sends data to ECHO / PLATO only (default)
                     1; sends data to FIS REL data tables in addition to ECHO/PLATO
                     
Uses CommonPAEData() for common XML fields with df.test_data.start_time as event date.

SendSMARTAttributes()

source code 

Sends SMART attribute sector data to PAE database.

SATA devices send data to PAE_SMART_ATTR using PAE_SMART_ATTR_ATTR and PAE_SMART_ATTR_COMN message. SCSI devices use PAE_SMART_ATTR_SINGLE via sm.SAS_SMART_Attributes() and sends extra RAW_HEX field.

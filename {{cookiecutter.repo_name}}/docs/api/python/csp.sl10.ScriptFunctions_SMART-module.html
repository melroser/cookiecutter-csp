ForceSMARTFrameSave()

source code 

Forces SMART frame save on SATA or SAS drives to ensure latest frame is updated. SAS uses rezero unit command while SATA uses SMART Factory command.

GetSmartFrameHeaderData(key='')

source code  

Retrieves SMART frame data at the given offset with given length. 
Issues a rezero unit command to force last SMART frame save if SAS.
Uses code from ParseSM2Log.py (by Michael Ullmann)

key: string:  SMART frame attribute to collect. This key must exist in 
                    SMART_FRAME_HEADER defined in ScriptFunctions_SMART.py

SmartCheck(enable_failure_threshold=True, fail_attr_list={}, use_default_fail=True, collect_data=True)

source code  

Check & display SMART attributes, fail drive if IOEDC or Pending Count > Zero, can add more attributes with different failing thresholds

enable_failure_threshold: boolean:  True (default) to fail when tripped, or false to avoid failure of drive.
fail_attr_list: dictionary:         Custom list of attributes to fail on - ex [attribute name, threshold]} or {atribute #:threshold}
use_default_fail: boolean:          True (default) uses default device thresholds for P-list & IOEDC, false otherwise
collect_data: boolean:              True (default) collects SMART attribute log and sends them to PAE_SMART_ATTR_ATTR

GetSmartPowerOnHours(printTS=0)

source code 

Returns SMART power-on hours for SATA and SAS drives. (SATA - From Smart Attribute #9, SAS - from Log Page 0x3E) Will return -1 by default if there is a problem pulling POH.

For UDS time stamp, use GetUDSTimeStamp() instead.

printTS: boolean: If set to 1, will display POH in the result file. (default 0)

TestFAFH(logToECHO=True)

source code  

Collects production FAFH data. Grabs data from the most recent FAFH log
frame. If CollectVerboseFAFHData is set in test options, will grab data from
all available log frames. 
    
Inputs:    
    logToEcho: boolean:    Sends data to PAE_FAFH_STD/DEV if true (default yes)

RaiseException(label, ec)

source code 

Raise a CSP script exception, indicating a CSP or drive failure, rather than a Python or main_cell failure.

label: String describing the failure. ec: integer: Error code to raise. EC_BASE will be added to the error code

ScriptErrorHandler(label, ec, clear_Gemini_SN=True)

source code  

Perform error handling sequence to gather drive data after a test failure.
Will power cycle the drive.

label: string describing the error that occurred.
ec: the error code of the failure
clear_Gemini_SN: boolean: optional: whether to clear a Gemini's host 
    database of the current drive's serial number.  Used to prevent cell
    errors from reporting an error for the previous drive when there is 
    no robotic gantry to update the host database before the test.

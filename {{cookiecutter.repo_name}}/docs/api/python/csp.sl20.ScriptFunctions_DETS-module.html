_DETS_Check_Status(functionName, allowNoSupport=True)

source code  

Common function used internally to check DETS return status.
called after a dits.RetrieveSDBP() to verify status.

functionName: string
    value returned from callers misc.GetFunctionName()
allowNoSupport: boolean
    if false will throw exception on unsupported response.
    If true, prints message to log with no exception.

Returns 0 if successful, -1 if DETS is not supported

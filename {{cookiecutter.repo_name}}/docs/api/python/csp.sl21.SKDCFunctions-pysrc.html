 19      This function determines if current drive is SKDC Unique Drive (M8 or M9T). 
 20      Returns true if so, false otherwise

 22      '''  
 23          Model numbers and FW revs in use by M8 and M9 
 24       
 25          M8BP1 500GB 
 26          Model : ST500LM012 HN-M500MBB 
 27          F/W Rev : 2AR20004 
 28           
 29          M8BP1 1TB  
 30          Model : ST1000LM024 HN-M101MBB 
 31          F/W Rev : 2AR20003 
 32           
 33          M8BP2 500GB 
 34          Model : ST500LM012 HN-M500MBB 
 35          F/W Rev : 2BA30001 
 36           
 37          M8BP2 1TB  
 38          Model : ST1000LM024 HN-M101MBB 
 39          F/W Rev : 2BA30001 
 40           
 41          M9T 2TB  
 42          Model : ST2000LM003 HN-M201RAD 
 43          F/W Rev : 2BC10001 
 44           
 45          M9T 1.5TB  
 46          Model : S30324T1500LM006 HN-M151RAD 
 47          F/W Rev : 2BC10001 
 48      ''' 
 49      if (df.test_data.SKDCunique == None): 
 50          df.test_data.SKDCunique = False 
 51           
 52          sf.ConfigurePort() 
 53          fwRev = misc.byteToString(23*2, 4*2, 'ascii', -1)[:2] 
 54           
 55          serialNum = sf.GetSerialNumber() 
 56          modelNum = sf.GetModelNumber() 
 57           
 58          # 1. SKDC drives always have 14 digit serial numbers 
 59          # 2. Look for SKDC unique string (HN-) in model number 
 60          # 3. Look for 2A or 2B as beginning string in FW rev 
 61          if ('HN-M' in modelNum) and (len(serialNum) == 14) and (fwRev in df.test_data.SKDCfwTag): 
 62              df.test_data.SKDCunique = True 
 63           
 64      return df.test_data.SKDCunique

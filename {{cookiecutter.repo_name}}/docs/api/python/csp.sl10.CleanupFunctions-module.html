ClearSMART(useDITS=0, doDITSunlock=False)

source code  

Reinitializes SMART. Default method is to reset via serial port if no argument is supplied.
DITS method will be attempted if it fails to reset via serial port.
If NoSerialPort option is selected, then resetMethod will be overridden to 1 (through DITS) 

useDITS     : integer: 0: resets SMART using serial port (default - SATA and LCO SAS only)
                       1: resets SMART using DITS (optional, and automatically attempted if 0 fails)
doDITSunlock: boolean: False: skips DITS unlock command (assumes already unlocked - default)
                       True:  issues DITS unlock in case it is not unlocked

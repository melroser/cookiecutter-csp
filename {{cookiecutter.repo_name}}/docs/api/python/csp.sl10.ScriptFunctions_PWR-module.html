SetChamberCurrent(i5, i12)

source code 

Set chamber power supply current. Power supply will shut down if this value is exceeded. Note the default setting is set by hostman with the values from the environment max current boxes. For hp GPIB this is IOUT.

i5: float: amps i12: float: amps

SetChamberPwrLimits(vLimit_5, iLimit_5, vLimit_12, iLimit_12)

source code 

Set chamber power supply limits. Programs the upper limit value that the power supply will accept. For hp GPIB this is IMAX, VMAX

iLimit_5: float: amps vLimit_5: float: volts iLimit_12: float: amps vLimit_12: float: volts

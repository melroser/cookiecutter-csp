BER_By_Head(TrackMap, ODTestTrack, SideTracks)

source code 

For each head, it performs 1 pass write on one selected track (ODTestTrack), then 1000 reads on the same track. For SAS, BER data is calculated from log pages. For SATA, BER is collected through serial port. It also sends out an entry to DRV_FDT_LEV table with drive state data only.

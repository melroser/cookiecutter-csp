DisableEPCTimer(EPCMode, saveSetting=0)

source code  

Disable particular EPC Timer.

Args:
    EPCMode: an int or hex which should be equivalent to the following:
       0x00 for STANDBY / STANDBY_Z,
       0x81 for EPC_IDLE_A,
       0x82 for EPC_IDLE_B
       0x83 for EPC_IDLE_C
       0x01 for STANDBY_Y
    saveSetting: an int, either 0 for False or 1 for True.
                 If 1/True settings are persistent across power-on resets

Returns:
    None

CHS2LBA(cyl, hd, sector, doDITSunlock=False)

source code  

CHS2LBA
Translates Cylinder head sector information to LBA address

Inputs:
    cyl - int, cylinder
    hd - int, head
    sector - int, sector
    doDITSunlock - boolean, whether to send a DITS unlock command. Defaults
                   to false since it is assumed that DITS unlock should have
                   already been called when the drive spins up.
Outputs:
    lba - int, the LBA corresponding to the CHS

LBA2CHS(lba, doDITSunlock=False)

source code  

LBA2CHS
Translates LBA to cylinder head and sector

Inputs:
    lba - int, LBA address
    doDITSunlock - boolean, whether to send a DITS unlock command. Defaults
                   to false since it is assumed that DITS unlock should have
                   already been called when the drive spins up.
Outputs:
    cyl, hd, sector
    cyl - the cylinder corresponding to the given lba
    hd - the head corresponding with the given lba
    sector - the sector corresponding to the given lba

SelfTestOffline(self_test_code, timeout_limit, poll_freq, captive=True)

source code 

Send SCSI DST short or long to the device. Since Seagate does not support captive (foreground) DST modes, it is simulated by not returning control back to the main thread.

self_test_code: integer: 1 for short test, 2 for extended test timeout_limit: integer: Timeout limit of this test poll_freq: integer: Frequency (in seconds) to check DST status

SendDiagnostic(self_test_bit, self_test_code, pf=0, dev_offl=0, unit_offl=0, param_list_len=0, test_name='')

source code  

Issue Send Diagnostic command to SCSI device to start drive self-test.

self_test_bit: integer:  Self-Test bit. If set to 1, it will cause Self-Test code to be 0.
                         With this bit set to 1, it will perform a default logic unit self-test.
                         If set to 0, it will perform self-test defined by self_test_code.
self_test_code: integer: Self-Test code. 
                            0: This value is used when Self-Test bit is 1.
                            1: run DST short (background)
                            2: run DST long (background)
                            3: reserved
                            4: abort DST
                            5: run DST short (foreground / captive) - not supported by STX
                            6: run DST long (foreground / captive) - not supported by STX
                            7: reserved
pf: integer:              Page format bit. 0 specifies that parameters are vendor specific.
                          (Default = 0)
dev_offl: boolean:        Device Offline bit. If set to 1, DST may perform diag commands that
                          can affect logical units in the SCSI device i.e. reservations,
                          log parameters, sense data etc. (Default = 0)
unit_offl: boolean:       Unit Offline bit. If set to 1, DST may perform diag commands that can
                          affect user accessible medium. (Default = 0)
param_list_len: integer   Data length that will be sent to the device.
test_name: string:        Test name for debug purposes if the command fails.

__readWaveformData(waveformDataFilename)

source code  

Read in waveform data file in Perseus software format

Input example:
    0.0=3000.0,2000.0,1000.0,
    0.1=2000.0,3000.0,,
    0.5=5000.0,,,
    
Output example:
    {0: {0.0: 3000.0, 0.1: 2000.0, 0.5: 5000.0}, 1: {0.0: 2000.0, 0.1: 3000.0}, 2: {0.0: 1000.0}, 3: {}}
        A hash with one key per channel, the value is a hash of timestamps to float voltages

__processWaveformData(waveformData)

source code  

Interpolate waveform data for every WAVEFORM_RATE milliseconds.

Input data example:
    {0: {0.0: 3000.0, 0.1: 2000.0, 0.5: 5000.0}, 1: {0.0: 2000.0, 0.1: 3000.0}, 2: {0.0: 1000.0}, 3: {}}
        A hash with one key per channel, the value is a hash of timestamps to float voltages
    
Output data example:
    {0: [3000, 2000, 2750, 3500, 4250, 5000], 1: [2000, 3000], 2: [1000], 3: []}
        A hash with one key per channel, the value is a list of integer voltage samples WAVEFORM_RATE apart
